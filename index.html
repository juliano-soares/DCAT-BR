Sobre o DCAT-BR

O DCAT-BR é um perfil de aplicação baseado no vocabulário Data Catalogue Vocabulary (DCAT) da W3C, desenvolvido especificamente para descrever catálogos de dados públicos no Brasil. Seu objetivo principal é facilitar a descoberta e o reuso de dados governamentais brasileiros.

O perfil foi desenvolvido no contexto do Catálogo Nacional de Dados, uma iniciativa do governo brasileiro para centralizar e padronizar a descrição de conjuntos de dados públicos.

Como Contribuir

Problemas encontrados ou sugestões para novas funcionalidades podem ser submetidos como issues no repositório do DCAT-BR no GitHub.

Para contribuir com o desenvolvimento:

  1. Faça um fork do repositório
  2. Crie uma branch para sua contribuição
  3. Faça suas alterações e testes
  4. Submeta um Pull Request
