Resumo

Este documento apresenta os vocabulários controlados utilizados no perfil DCAT-BR para padronizar a descrição de metadados em catálogos de dados públicos no Brasil. Estes vocabulários garantem consistência semântica e facilitam a interoperabilidade entre diferentes portais e agregadores de dados governamentais.

Status deste documento

Este documento é mantido pelo Grupo de Trabalho DCAT-BR e está sujeito a atualizações conforme novos vocabulários são adicionados ou vocabulários existentes são revisados. Comentários e propostas de alteração devem ser enviados por meio do repositório público indicado na seção de governança.

Introdução

Os vocabulários controlados são conjuntos de termos padronizados utilizados para garantir consistência na descrição de metadados em catálogos de dados. No contexto do DCAT-BR, estes vocabulários são essenciais para promover a interoperabilidade semântica entre diferentes portais e plataformas de dados governamentais.

Cada vocabulário listado neste documento fornece um conjunto de valores permitidos para campos específicos dos metadados, tais como frequência de atualização, licenças de uso, temas, localização geográfica, entre outros. Esta padronização facilita a descoberta, agregação e reuso de dados governamentais em escala nacional.
