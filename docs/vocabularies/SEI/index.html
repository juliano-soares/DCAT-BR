Resumo

O Vocabulário Controlado para Observância Legal (SEI) é usado para indicar a observância legal de uma distribuição de dados em catálogos de dados públicos no Brasil. Este vocabulário controlado baseia-se no Sistema Eletrônico de Informações (SEI) e define termos padronizados para descrever se os dados são públicos, restritos ou sigilosos, conforme a legislação brasileira aplicável.

Status deste documento

Este documento descreve o vocabulário controlado para Observância Legal (SEI) mantido pelo Grupo de Trabalho DCAT-BR. Comentários e propostas de alteração devem ser enviados por meio do repositório público indicado na seção de governança.

Introdução

O campo Observância Legal é uma propriedade essencial no perfil DCAT-BR para descrever a observância legal de uma distribuição de dados. Este campo utiliza a propriedade dct:accessRights do vocabulário Dublin Core Terms e deve referenciar um identificador de observância legal conforme o Sistema Eletrônico de Informações (SEI).

Este vocabulário classifica os dados em três categorias principais:

  • Público: Dados sem restrições de acesso
  • Restrito: Dados com restrições de acesso baseadas em legislação específica
  • Sigiloso: Dados com sigilo baseado em legislação específica

Selo de Conformidade: Bronze
Base: SEI
URI Base: https://dcat-br.gov.br/vocabularies/SEI/esquema

O vocabulário está disponível nos formatos Turtle, RDF/XML e JSON-LD (consulte a seção Formatos Disponíveis).

Exemplo de Uso

Abaixo estão exemplos de como usar o campo Observância Legal em distribuições DCAT-BR: 

@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix dct: <http://purl.org/dc/terms/> .

<https://exemplo.gov.br/dataset/1/distribution/1>
    a dcat:Distribution ;
    dct:title "Dados Públicos em CSV"@pt-BR ;
    dct:accessRights <https://dcat-br.gov.br/vocabularies/SEI/1> ;
    dcat:downloadURL <https://exemplo.gov.br/dados.csv> .
    
@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix dct: <http://purl.org/dc/terms/> .

<https://exemplo.gov.br/dataset/1/distribution/2>
    a dcat:Distribution ;
    dct:title "Dados Restritos"@pt-BR ;
    dct:accessRights <https://dcat-br.gov.br/vocabularies/SEI/3> ;
    dct:license <https://dcat-br.gov.br/vocabularies/VCR-LU/USO_RESTRITO> ;
    dcat:downloadURL <https://exemplo.gov.br/dados-restritos.csv> .
