Resumo

O campo Tema Associado ao Conjunto de Dados é usado para classificar e categorizar os temas dos conjuntos de dados em catálogos de dados públicos no Brasil. Este vocabulário controlado baseia-se no Vocabulário Controlado do Governo Eletrônico - VCGE (Versão 2.1.0 - 2016) e define termos padronizados para descrever os temas dos conjuntos de dados, facilitando a descoberta e o uso de dados governamentais brasileiros através de uma classificação temática consistente.

Status deste documento

Este documento descreve o vocabulário controlado para Tema Associado ao Conjunto de Dados mantido pelo Grupo de Trabalho DCAT-BR. Comentários e propostas de alteração devem ser enviados por meio do repositório público indicado na seção de governança.

Introdução

O campo Tema Associado ao Conjunto de Dados é uma propriedade essencial no perfil DCAT-BR para classificar e categorizar os temas dos conjuntos de dados. Este campo utiliza a propriedade dcat:theme do vocabulário DCAT e permite associar um ou mais temas a cada conjunto de dados, facilitando a descoberta através de classificação temática.

Este vocabulário é um subconjunto de temas (top concepts) utilizado no Portal Brasileiro de Dados Abertos (dados.gov.br), alinhado ao VCGE versão 2.1.0 de 2016. O VCGE é o vocabulário oficial do governo brasileiro para classificação de informações governamentais.

Base: Vocabulário Controlado do Governo Eletrônico - VCGE (Versão 2.1.0 - 2016)

Exemplo de Uso

Abaixo está um exemplo de como usar o campo Tema Associado ao Conjunto de Dados em um conjunto de dados DCAT-BR: 

@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix dct: <http://purl.org/dc/terms/> .

<https://exemplo.gov.br/dataset/1>
    a dcat:Dataset ;
    dct:title "Dados de Educação"@pt-BR ;
    dcat:theme <http://vocab.e.gov.br/id/governo/2016/EDUCACAO> .
