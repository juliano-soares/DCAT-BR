Resumo

O Vocabulário Controlado para Tipo de Recurso é usado para classificar o tipo de recurso disponibilizado em uma distribuição de dados em catálogos de dados públicos no Brasil. Este vocabulário controlado personalizado define termos padronizados para categorizar diferentes tipos de recursos, facilitando a descoberta e o uso de dados governamentais brasileiros.

Status deste documento

Este documento descreve o vocabulário controlado para Tipo de Recurso mantido pelo Grupo de Trabalho DCAT-BR. Comentários e propostas de alteração devem ser enviados por meio do repositório público indicado na seção de governança.

Introdução

O campo Tipo de Recurso é uma propriedade utilizada no perfil DCAT-BR para classificar o tipo de recurso disponibilizado em uma distribuição. Este campo utiliza a propriedade dct:type do vocabulário Dublin Core Terms e permite categorizar recursos como dados, APIs, documentação, entre outros.

Este vocabulário foi desenvolvido especificamente para o contexto brasileiro, considerando as necessidades de catalogação de recursos de dados governamentais. A classificação por tipo de recurso facilita a descoberta e o acesso adequado aos diferentes tipos de conteúdo disponibilizados pelos órgãos públicos.

Selo de Conformidade: Bronze
Base: Personalizado
URI Base: https://dcat-br.github.io/dcat-br/vocabularies/tipo-recurso/

O vocabulário está disponível nos formatos Turtle (TTL), RDF/XML e JSON-LD (consulte a seção Formatos Disponíveis).

Termos do Vocabulário

O vocabulário Tipo de Recurso inclui os seguintes termos padronizados:

Termo Notação URI Descrição Formatos
Dados DADOS https://dcat-br.github.io/dcat-br/vocabularies/tipo-recurso/DADOS Recurso do tipo dados, contendo informações estruturadas ou não estruturadas. Turtle | RDF
Dicionário de Dados DICIONARIO_DE_DADOS https://dcat-br.github.io/dcat-br/vocabularies/tipo-recurso/DICIONARIO_DE_DADOS Dicionário de dados que descreve a estrutura e o significado dos campos de um conjunto de dados. Turtle | RDF
Documentação DOCUMENTACAO https://dcat-br.github.io/dcat-br/vocabularies/tipo-recurso/DOCUMENTACAO Documentação relacionada ao conjunto de dados, incluindo manuais, guias e especificações. Turtle | RDF
API API https://dcat-br.github.io/dcat-br/vocabularies/tipo-recurso/API Interface de Programação de Aplicações (API) para acesso programático aos dados. Turtle | RDF
Outro OUTRO https://dcat-br.github.io/dcat-br/vocabularies/tipo-recurso/OUTRO Outro tipo de recurso não especificado nas categorias anteriores. Turtle | RDF

Exemplo de Uso

Abaixo estão exemplos de como usar o campo Tipo de Recurso em distribuições DCAT-BR: 

@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix dct: <http://purl.org/dc/terms/> .

<https://exemplo.gov.br/dataset/1/distribution/1>
    a dcat:Distribution ;
    dct:title "Dados em CSV"@pt-BR ;
    dct:type <https://dcat-br.github.io/dcat-br/vocabularies/tipo-recurso/DADOS> ;
    dcat:downloadURL <https://exemplo.gov.br/dados.csv> .
    
@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix dct: <http://purl.org/dc/terms/> .

<https://exemplo.gov.br/dataset/1/distribution/2>
    a dcat:Distribution ;
    dct:title "API REST"@pt-BR ;
    dct:type <https://dcat-br.github.io/dcat-br/vocabularies/tipo-recurso/API> ;
    dcat:accessURL <https://api.exemplo.gov.br/v1/dados> .
