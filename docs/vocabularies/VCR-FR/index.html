Resumo

O Vocabulário Controlado para Frequência (VCR-FR) é usado para indicar a frequência de atualização dos dados em catálogos de dados públicos no Brasil. Este vocabulário controlado baseia-se no Collection Description Frequency Namespace do Dublin Core e define termos padronizados para descrever com que frequência um conjunto de dados é atualizado, facilitando a descoberta e o uso de dados governamentais brasileiros.

Status deste documento

Este documento descreve o vocabulário controlado VCR-FR mantido pelo Grupo de Trabalho DCAT-BR. Comentários e propostas de alteração devem ser enviados por meio do repositório público indicado na seção de governança.

Introdução

O campo Frequência (VCR-FR) é um vocabulário controlado utilizado no perfil DCAT-BR para padronizar a descrição da periodicidade de atualização de conjuntos de dados. Ao utilizar termos padronizados, os catálogos de dados garantem consistência semântica e facilitam a descoberta de dados baseada em critérios de atualização.

Este vocabulário é baseado nas especificações do Dublin Core Metadata Initiative (DCMI) e foi adaptado para o contexto brasileiro, mantendo compatibilidade com padrões internacionais de catalogação de dados.

Termos do Vocabulário

O vocabulário VCR-FR inclui os seguintes termos de frequência padronizados:

Termo Descrição
Anual O evento ocorre uma vez por ano
Bienal O evento ocorre a cada dois anos
Bimestral O evento ocorre a cada dois meses
Quinzenal O evento ocorre a cada duas semanas
Contínuo O evento se repete de forma contínua, sem interrupção
Diária O evento ocorre uma vez por dia
Irregular O evento ocorre em intervalos irregulares
Mensal O evento ocorre uma vez por mês
Trimestral O evento ocorre a cada três meses
Semestral O evento ocorre duas vezes por ano
Semimensal O evento ocorre duas vezes por mês
Semissemanal O evento ocorre duas vezes por semana
Três vezes por mês O evento ocorre três vezes por mês
Três vezes por semana O evento ocorre três vezes por semana
Três vezes ao ano O evento ocorre três vezes por ano
Trienal O evento ocorre a cada três anos
Semanal O evento ocorre uma vez por semana
