Resumo

O Vocabulário Controlado para Cobertura Espacial (VCR-CE) é usado para indicar a cobertura espacial de uma distribuição de dados em catálogos de dados públicos no Brasil. Este vocabulário controlado define termos padronizados para descrever a abrangência territorial dos dados, classificando-os em níveis federal, estadual e municipal.

Status deste documento

Este documento descreve o vocabulário controlado para Cobertura Espacial (VCR-CE) mantido pelo Grupo de Trabalho DCAT-BR. Comentários e propostas de alteração devem ser enviados por meio do repositório público indicado na seção de governança.

Introdução

O campo Cobertura Espacial é uma propriedade essencial no perfil DCAT-BR para descrever a abrangência territorial de uma distribuição de dados. Este campo utiliza a propriedade dct:spatial do vocabulário Dublin Core Terms e deve referenciar um identificador de cobertura espacial conforme este vocabulário.

Este vocabulário classifica a cobertura espacial dos dados em três níveis administrativos:

  • Federal: Dados de abrangência nacional
  • Estadual: Dados de abrangência estadual
  • Municipal: Dados de abrangência municipal

Selo de Conformidade: Bronze
Base: Personalizado
URI Base: https://dcat-br.gov.br/vocabularies/VCR-CE/esquema

O vocabulário está disponível nos formatos Turtle, RDF/XML e JSON-LD (consulte a seção Formatos Disponíveis).

Termos do Vocabulário

O vocabulário VCR-CE inclui os seguintes termos de cobertura espacial:

Termo Notação URI Descrição
Federal FEDERAL https://dcat-br.gov.br/vocabularies/VCR-CE/FEDERAL Cobertura espacial de abrangência federal, referente a dados que se aplicam a todo o território nacional brasileiro.
Estadual/Distrital ESTADUAL https://dcat-br.gov.br/vocabularies/VCR-CE/ESTADUAL Cobertura espacial de abrangência estadual ou distrital, referente a dados que se aplicam a um ou mais estados brasileiros ou ao Distrito Federal.
Municipal MUNICIPAL https://dcat-br.gov.br/vocabularies/VCR-CE/MUNICIPAL Cobertura espacial de abrangência municipal, referente a dados que se aplicam a um ou mais municípios brasileiros.

Exemplo de Uso

Abaixo estão exemplos de como usar o campo Cobertura Espacial em distribuições DCAT-BR: 

@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix dct: <http://purl.org/dc/terms/> .

<https://exemplo.gov.br/dataset/1/distribution/1>
    a dcat:Distribution ;
    dct:title "Dados Nacionais"@pt-BR ;
    dct:spatial <https://dcat-br.gov.br/vocabularies/VCR-CE/FEDERAL> ;
    dcat:downloadURL <https://exemplo.gov.br/dados-nacionais.csv> .
    
@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix dct: <http://purl.org/dc/terms/> .

<https://exemplo.gov.br/dataset/1/distribution/2>
    a dcat:Distribution ;
    dct:title "Dados Estaduais"@pt-BR ;
    dct:spatial <https://dcat-br.gov.br/vocabularies/VCR-CE/ESTADUAL> ;
    dcat:downloadURL <https://exemplo.gov.br/dados-estaduais.csv> .
    
@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix dct: <http://purl.org/dc/terms/> .

<https://exemplo.gov.br/dataset/1/distribution/3>
    a dcat:Distribution ;
    dct:title "Dados Municipais"@pt-BR ;
    dct:spatial <https://dcat-br.gov.br/vocabularies/VCR-CE/MUNICIPAL> ;
    dcat:downloadURL <https://exemplo.gov.br/dados-municipais.csv> .
