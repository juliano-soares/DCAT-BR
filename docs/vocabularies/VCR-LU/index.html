Resumo

O Vocabulário Controlado para Licença de Uso (VCR-LU) é usado para indicar a licença sob a qual uma distribuição de dados está disponível em catálogos de dados públicos no Brasil. Este vocabulário controlado baseia-se em licenças Creative Commons e Open Data Commons.

Status deste documento

Este documento descreve o vocabulário controlado para Licença de Uso (VCR-LU) mantido pelo Grupo de Trabalho DCAT-BR. Comentários e propostas de alteração devem ser enviados por meio do repositório público indicado na seção de governança.

Introdução

O campo Licença de Uso é uma propriedade essencial no perfil DCAT-BR para descrever a licença sob a qual uma distribuição de dados está disponível. Este campo utiliza a propriedade dct:license do vocabulário Dublin Core Terms e deve referenciar uma licença pública.

Este vocabulário inclui as principais licenças utilizadas para dados abertos, incluindo licenças Creative Commons e Open Data Commons.

Selo de Conformidade: Bronze
Base: Creative Commons + Open Data Commons
URI Base: https://dcat-br.gov.br/vocabularies/VCR-LU/esquema

O vocabulário está disponível nos formatos Turtle, RDF/XML e JSON-LD (consulte a seção Formatos Disponíveis).

Termos do Vocabulário

O vocabulário VCR-LU inclui as seguintes licenças:

Termo Notação URI Descrição Documentação
Creative Commons CCZero cc-zero https://creativecommons.org/publicdomain/zero/1.0/ Dedicado ao domínio público, permitindo uso, modificação e distribuição sem restrições, sem necessidade de atribuição. CC0 | SPDX
Creative Commons Atribuição cc-by https://creativecommons.org/licenses/by/4.0/ Permite uso comercial e não comercial, modificação e distribuição, desde que seja fornecida atribuição ao autor original. CC-BY-4.0 | SPDX
Licença Aberta para Bases de Dados (ODbL) do Open Data Commons odc-odbl https://opendatacommons.org/licenses/odbl/1-0/ Licença específica para bancos de dados abertos que permite uso, modificação e distribuição, desde que seja mantida a atribuição, compartilhamento de alterações e uso da mesma licença para trabalhos derivados. ODbL | SPDX
Licença e Dedicação ao Domínio Público do Open Data Commons (PDDL) odc-pddl https://opendatacommons.org/licenses/pddl/1-0/ Dedicação ao domínio público para dados, permitindo uso, modificação e distribuição sem restrições. PDDL | ODC

Exemplo de Uso

Abaixo estão exemplos de como usar o campo Licença de Uso em distribuições DCAT-BR: 

@prefix dcat: <http://www.w3.org/ns/dcat#> .
@prefix dct: <http://purl.org/dc/terms/> .

<https://exemplo.gov.br/dataset/1/distribution/1>
    a dcat:Distribution ;
    dct:title "Dados em CSV"@pt-BR ;
    dct:license <https://creativecommons.org/licenses/by/4.0/> ;
    dcat:downloadURL <https://exemplo.gov.br/dados.csv> .
