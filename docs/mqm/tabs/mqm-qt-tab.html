Análise Quantitativa (MQM-QT)

O propósito deste relatório é analisar a qualidade dos metadados (dados que descrevem outros dados) em portais de dados abertos, utilizando o Framework de Metodologia de Qualidade em Metadados (MQM).

Metodologia de Avaliação

A avaliação é feita com base em métricas de qualidade, que são classificadas em cinco dimensões:

  • Encontrabilidade: Facilita a descoberta dos dados através de mecanismos de busca
  • Acessibilidade: Garante que os dados possam ser acessados e recuperados
  • Interoperabilidade: Permite a integração e combinação de dados de diferentes fontes
  • Reutilizabilidade: Facilita o reuso dos dados para diferentes propósitos
  • Contextualidade: Fornece informações contextuais sobre os dados

Em cada dimensão é avaliado um conjunto de indicadores que são classificados em pontos, totalizando a pontuação máxima de cada dimensão.

Encontrabilidade (Findable) - Máximo: 100 pontos

Avalia a eficácia com que os metadados ajudam pessoas e máquinas a encontrar os conjuntos de dados.

Indicador Descrição Peso
Utilização de Palavra-chave Verifica se dcat:keyword está definido. 20
Tema Verifica se dcat:theme foi atribuído, auxiliando na exploração temática. 30
Vocabulário de Tema (VCR) Verifica se algum dos Vocabulários Controlados de Referência (VCR) foi usado para o tema. VCR-Temas 10
Pesquisa Espacial Verifica se dcat:spatial foi definido, permitindo a pesquisa por localização. 20
Pesquisa Temporal Verifica se metadados temporais (e.g., dct:temporal) foram definidos. 20

Acessibilidade (Accessible) - Máximo: 100 pontos

Garante que o conjunto de dados possa ser efetivamente acessado. Verifica a existência e a funcionalidade dos links.

Indicador Descrição Peso
Acessibilidade da URL de Acesso Verifica se a URL de acesso (dcat:accessURL) responde corretamente (código 200-300). 50
URL de Transferência Verifica se a URL de transferência (dcat:downloadURL) é um link direto para o download. 50

Interoperabilidade (Interoperable) - Máximo: 110 pontos

Avalia a capacidade de os dados serem combinados e utilizados por diferentes aplicações (formatos padronizados e legibilidade por máquina).

Indicador Descrição Peso
Formato Verifica se o formato do arquivo (dct:format) está definido. 50
Tipo de Mídia Verifica se o tipo de mídia (dcat:mediaType) está definido. 10
Formato/Tipo de Mídia de Vocabulário Verifica se pertencem a um vocabulário controlado (e.g., lista IANA). VCR-Formatos 10
Não Proprietário Verifica se o formato da distribuição é não proprietário. 20
Legível por Máquina Verifica se o formato é legível por máquina. 20
Conformidade com o DCAT-BR Valida se os metadados estão em conformidade com as regras do DCAT-BR (via serviço de validação SHACL). SHACL-DCAT-BR 30

Reutilizabilidade (Reusable) - Máximo: 75 pontos

Depende da clareza das condições de uso dos dados (licenças, restrições e pontos de contato).

Indicador Descrição Peso
Informações da Licença Verifica se uma licença (dct:license) foi definida, essencial para a reutilização. 20
Vocabulário da Licença Incentiva o uso de VCT para licenças para evitar informações incorretas. VCR-LU 10
Restrições de Acesso Verifica se os direitos de acesso (dct:accessRights) estão definidos (público ou restrito). 10
Vocabulário das Restrições de Acesso Verifica se o VCT para direitos de acesso é utilizado. VCR-SEI 5
Ponto de Contato Verifica se o ponto de contato (dct:contactPoint) para dúvidas está definido. 20
Publicador Verifica se a organização/pessoa que publicou os dados (dct:publisher) está definida. 10

Contextualidade (Contextuality) - Máximo: 15 pontos

Mede a presença de metadados que fornecem contexto adicional, facilitando a compreensão e o escopo dos dados.

Indicador Descrição Peso
Tamanho do Arquivo Especifica o tamanho do arquivo em bytes (dct:byteSize). 5
Data de Emissão A data em que o conjunto de dados foi divulgado (dct:issued). 5
Data de Alteração A data em que o conjunto de dados foi alterado pela última vez (dct:modified). 5

Classificação da Qualidade (Selo)

Com base na pontuação total obtida nas cinco dimensões (máximo de 405 pontos), a MQM classifica a qualidade dos metadados:

Notação Escala de Pontos
Excelente – Selo Ouro 351 - 405
Bom – Selo Prata 221 - 350
Suficiente – Selo Bronze 121 - 220
Insuficiente – Sem Selo 0 - 120

Selos por Dimensão (FAIR+C)

Para cada dimensão que esteja completa, ou seja, com o máximo de pontuação possível, é emitido um selo representando o princípio em questão:

Dimensão Selo Pontuação Máxima Critério para Obtenção
Encontrabilidade (E) E 100 pontos Obtém o selo quando alcança 100 pontos na dimensão de Encontrabilidade
Acessibilidade (A) A 100 pontos Obtém o selo quando alcança 100 pontos na dimensão de Acessibilidade
Interoperabilidade (I) I 110 pontos Obtém o selo quando alcança 110 pontos na dimensão de Interoperabilidade
Reutilizabilidade (R) R 75 pontos Obtém o selo quando alcança 75 pontos na dimensão de Reutilizabilidade
Contextualidade (C) C 15 pontos Obtém o selo quando alcança 15 pontos na dimensão de Contextualidade
