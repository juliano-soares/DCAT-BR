Metodologia de Qualidade em Metadados (MQM)

A Metodologia de Qualidade em Metadados (MQM) é uma abordagem abrangente desenvolvida para avaliar e garantir a qualidade dos metadados no contexto do perfil DCAT-BR.

A MQM estabelece critérios e métricas para avaliar a qualidade dos metadados de conjuntos de dados públicos, considerando diferentes dimensões que impactam a capacidade de descoberta, acessibilidade, interoperabilidade, reutilização e contextualização dos dados.

Dimensões da Qualidade

A metodologia MQM avalia os metadados através de cinco dimensões principais:

  • Encontrabilidade: Facilita a descoberta dos dados através de mecanismos de busca
  • Acessibilidade: Garante que os dados possam ser acessados e recuperados
  • Interoperabilidade: Permite a integração e combinação de dados de diferentes fontes
  • Reutilizabilidade: Facilita o reuso dos dados para diferentes propósitos
  • Contextualidade: Fornece informações contextuais sobre os dados
